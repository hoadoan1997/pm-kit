# Competitive Analysis Framework

## Feature Comparison Matrix

Rate each feature per competitor using a consistent scale:

| Rating | Meaning |
|--------|---------|
| - | Not available |
| * | Basic/limited |
| ** | Good, meets expectations |
| *** | Best-in-class |

## Positioning Map Dimensions

Choose two axes that reveal strategic differences:
- **Price vs Feature Breadth** — most common, shows value positioning
- **Ease of Use vs Power** — reveals target user sophistication
- **Self-serve vs Enterprise** — shows go-to-market strategy
- **Generalist vs Specialist** — shows market focus

## SWOT Analysis

- **Strengths** — Internal advantages over competitors
- **Weaknesses** — Internal gaps competitors exploit
- **Opportunities** — External trends you can capitalize on
- **Threats** — External risks from competitor moves or market shifts

## Strategic Recommendation Types

1. **Differentiate** — Double down on unique strengths
2. **Catch up** — Close critical feature gaps
3. **Reposition** — Change target segment or messaging
4. **Partner** — Integrate where competitors are strong
5. **Ignore** — Deprioritize areas where competition is irrelevant to your users

## Common Pitfalls

- Comparing yourself only to weaker competitors
- Ignoring indirect competitors and substitutes
- Treating all features as equally important
- Outdated competitive data (refresh quarterly)
- Missing the "do nothing" alternative
